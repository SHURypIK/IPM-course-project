Для восстановления бд из резервной копии необходимо
1)Удалить и заново создать бд "ipmcourseproject"
2)Из папки в которй лежит скрипт запустить командную строку и ввести команду "mysql -u root -p ipmcourseproject < backup.sql"
3)Ввести пароль "root"
Признаком успешного восстановления будет пустая строка
